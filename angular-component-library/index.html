Angular Component Library

Install latest @angular/cli: 


        npm i -g @angular/cli@latest

Create a new angular project (if you don't have an existing one) 


        ng new my-project --prefix=myprj --style=scss

(optional) Add Angular Material: 


        ng add @angular/material

Generate a new library project: 


        ng g library my-project-lib --prefix=myprjlib --style=scss

(optional) Set the registry of your library to your local npm repo: In package.json add the following to the highest level of the json object:
"publishConfig": { "registry":"http://localhost:4873/" },

Build and deploy your library to test it: 


        ng build my-project-lib
        cd dist/my-project-lib
        npm publish

Create another new angular project as a test environment to import my-project-lib into: 


        ng new my-project-lib-test --prefix=mplt --style=scss
        cd my-project-lib-test

npm install my-project-lib from your local npm registry: 


        npm i -S my-project-lib --registry http://localhost:4873/

Import the basic component from your library into the test project app.module.ts: 


        import { MyProjectLibModule } from 'my-project-lib';
        ...
        imports: [
          ...,
          MyProjectLibModule
        ]

Show the basic component from your library in the test project app.component.html: 


        ...
        <myprjlib-my-project-lib></myprjlib-my-project-lib>

Run the test project and verify that the test component works (http://localhost:4200). You should see this where you injected the component into the html page: "my-project-lib works!" 


        npm start
